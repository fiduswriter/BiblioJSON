### Download for browser usage

To use BiblioJSON on a webpage, simply include the corresponding file(s) in the header. All functions will be available in global namespace: [import-bibtex.js](/import-bibtex.js), [export-csl.js](/export-csl.js), [export-biblatex.js](/export-biblatex.js) or [bibliojson.js](/bibliojson.js) for a bundle that contains all of them.

In more complex setups, use the version [hosted on NPMJS.com](https://www.npmjs.com/package/bibliojson) instead.
